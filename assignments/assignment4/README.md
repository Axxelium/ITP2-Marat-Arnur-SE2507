# Assignment 4: Algorithms — Sorting and Searching

## Description

Implement classic sorting and searching algorithms from scratch.

## Tasks

1. **Bubble Sort** — `bubble_sort(lst)`: sort a list in-place using bubble sort.
2. **Merge Sort** — `merge_sort(lst)`: return a new sorted list using merge sort.
3. **Binary Search** — `binary_search(lst, target)`: search a *sorted* list;
   return the index of `target` or `-1` if not found.
4. **Linear Search** — `linear_search(lst, target)`: search an unsorted list;
   return the index of the first occurrence or `-1` if not found.

## How to Run

```bash
python algorithms.py
```
