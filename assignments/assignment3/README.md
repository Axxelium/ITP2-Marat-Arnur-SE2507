# Assignment 3: Data Structures — Lists, Stacks, and Queues

## Description

Implement generic `Stack` and `Queue` data structures from scratch
using Python lists as the underlying container.

## Tasks

1. Implement a `Stack` class with:
   - `push(item)`, `pop()`, `peek()`, `is_empty()`, `size()` methods.
   - `pop()` / `peek()` raise `IndexError` on an empty stack.

2. Implement a `Queue` class with:
   - `enqueue(item)`, `dequeue()`, `front()`, `is_empty()`, `size()` methods.
   - `dequeue()` / `front()` raise `IndexError` on an empty queue.

3. Use your `Stack` to implement `is_balanced(expression)` — checks whether
   brackets `()`, `[]`, `{}` in a string are balanced.

## How to Run

```bash
python data_structures.py
```
